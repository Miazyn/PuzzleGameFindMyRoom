# PuzzleGameFindMyRoom
A small little puzzle game in which you collect books in order to find a secret room.
Base Controls:

WASD to move
Left Ctrl => Crouch

Tab => Open Menu

Right Click/E => Pick Up Boxes

Have fun playing around. The game is playable under 5 minutes.
